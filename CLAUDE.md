# Medicine Reminder

A simple static web app for tracking daily medicine schedules. Deployed via GitHub Pages.

## Stack

- Vanilla HTML/CSS/JS (no build step)
- localStorage for persistence
- Browser Notification API for reminders

## Files

| File | Purpose |
|---|---|
| `index.html` | Single-page app — all UI, styles, and logic |

## Deployment

Hosted on GitHub Pages from the `main` branch root. Push to `main` to deploy.
